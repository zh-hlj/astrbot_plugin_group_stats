# AstrBot 群聊活跃统计插件（兼容 __init__.py）

📊 自动记录群聊活跃度，每日推送昨日数据，支持 WebUI 勾选群号 & 时间。

## 一键安装
WebUI → 插件 → 安装插件 → 从链安装 → 把仓库地址复制到输入框中 → 安装即可

## 添加应用群
输入应用的群号 + 选择自动推送时间

## 群内命令
- `/昨日活跃`  返回昨天数据
- `/今日统计`  返回今天实时数据
- `/在线人数`  返回当前群聊成员总数

## 文件说明
- `__init__.py` 空文件，保证旧加载器也能识别
- 数据文件：`group_stats.db`（SQLite，插件目录内）

## 版本要求
AstrBot ≥ 3.4.0
